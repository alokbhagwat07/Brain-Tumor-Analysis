## Brain Tumor Analysis Dashboard

An interactive **Power BI Dashboard** designed to analyze brain tumor patient data and provide meaningful insights through visually appealing charts and KPIs. This project demonstrates how data visualization can support healthcare professionals in understanding patient demographics, tumor characteristics, and diagnosis trends.

## Project Overview

The **Brain Tumor Analysis Dashboard** transforms raw healthcare data into actionable insights using Power BI. The dashboard provides an intuitive interface for exploring patient statistics, tumor distribution, age analysis, diagnosis trends, and affected brain regions.

## Features

-  Total Patients Analysis
-  Average Patient Age
-  Average Brain Tumor Size
-  Annual Diagnosis Trend
-  Tumor Type Distribution
-  Top Affected Brain Regions
-  Age Group Analysis
-  Grade-wise Filtering (Grades 1–5)
-  Confirmed & Unconfirmed Case Analysis
-  MRI Image Integration
-  Interactive Filters and Slicers


## 🛠️ Tech Stack

- **Power BI Desktop**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Data Modeling**
- **Microsoft Excel / CSV Dataset**
  

## Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Patients | Total number of patients in the dataset |
| Average Age | Average age of diagnosed patients |
| Average Tumor Size | Average tumor size across patients |
| Diagnosis Trend | Year-wise diagnosis analysis |
| Tumor Distribution | Distribution of different tumor types |
| Brain Region Analysis | Most affected brain regions |
| Age Group Analysis | Patient distribution by age groups |
